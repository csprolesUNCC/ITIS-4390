A simple mobile grocery demo

This prototype explores what a cleaner, easier Whole Foods shopping flow could look like on a phone. You can walk through onboarding, browse categories, open product details, and step through a basic cart and checkout experience.

•

Browse: category chips and a simple product grid.

•

Details: pricing, tags, and basic product info pulled from a JSON dataset.

•

Flow: onboarding → home → shop → cart / profile / payment screens.

Open demo from start → Jump into shop view

Built with HTML, Tailwind CSS, and a small JSON-backed DataAPI for course work and portfolio use.
